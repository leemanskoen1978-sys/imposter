## 🔄 Hoe Updates installeren?

Als je updates wilt (als die uitkomen):

```bash
# Ga naar je project folder
cd koens-imposter-game

# Pull de latest changes
git pull origin main

# Install eventuele nieuwe dependencies
npm install

# Run!
npm run dev
```

Als je op Netlify/Vercel deployed: dat gebeurt automatisch! ✨
